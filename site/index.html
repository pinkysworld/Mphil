MPhil research project

Time-aware malware family classification from dynamic sandbox reports.

This website documents my MPhil project on malware family classification using dynamic analysis reports, with a particular focus on temporal drift, leakage control, feature fusion, and probability calibration.

The current version reflects the first complete round of pipeline runs completed on 21 March 2026 and serves as a compact record of the study design, experiment set, and early findings.

Research Questions

What the project is trying to answer

RQ1

Which feature views retain useful signal once evaluation is aligned with time rather than random shuffling?

RQ2

How much do performance claims change once obvious leakage paths and naming shortcuts are controlled?

RQ3

Does feature fusion remain helpful under chronological testing and rolling-origin evaluation?

RQ4

Are the resulting probability scores reliable enough for calibration and selective prediction analysis?

Current Findings

First conclusions from the completed run set

Fusion helps

The strongest global chronological baseline so far uses fused views with SGD rather than a single feature family alone.

Text views carry most of the signal

API tokens and artifact tokens remain the strongest feature views in the invariance and discriminability analysis.

Time still makes the task harder

Walk-forward macro-F1 varies substantially by month, ranging from 0.669 to 0.929 across the current windows.

Confidence can be improved

Calibration reduced expected calibration error sharply, even with only a modest change in headline classification scores.
